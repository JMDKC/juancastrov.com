About me

My greatest passion is creating interactive experiences using code.

When I'm not writing code, playing around on processing, browsing cute dogs, taking pictures, or listening to music, I'm a full time student at Uniandes, where I major in CS and minor in Design. Feel free to reach me out on social media, or email me.

Projects

  • >> Processing experiments

    Mostly while experimenting on Processing for my Interactive Media Design class, I ended up designing and coding quite a few small, self-contained interactive 'games' or animations, exploring techniques such as image processing, pixel sorting, generative design and cellular automata.

    > Check it out on Github
  • >> Online portfolio - Juliana C.V.

    When her old webpage was going stale, my sister and I took upon creating a new online home for her work on a less restrictive platform with a responsive layout. It was prototyped, built and deployed using Wordpress on a VPS, reaching almost 10K page views within 3 months. I learnt a great deal about PHP, the linux shell and server administration.

    > julianacastrov.com
  • >> BADARTE

    As a project for our Web Design class, me and a couple of friends designed a new website proposal for the Banco de Archivos Digitales at Uniandes. We got such great feedback on the project that we were hired to build a fully functional prototype, using Omeka CMS, to serve as a template for future development.

    > See theme source on Github
